A school project about file management program over socket. Using TCP length prefix messages with client-server model

## Features


- Unsecure zero encryption.
- Cross-platform.
- Only ipv4.
- Doesn't compress data.

## Run
Python 3.5 or above is required.

Start server:

```
py server.py
```

Start client:

```
py client.py
```
